# Flight Search API

# Installation

* Assumes Python 2.7*


1. `git clone https://bitbucket.org/bradpugh/hipflights.git`
or https://github.com/pughbrad/hipflights.git

2. `cd hipflights`
3. `python setup.py develop`

# Run

- python -m search.api

# Endpoint

- GET http://localhost:8000/flights/search
